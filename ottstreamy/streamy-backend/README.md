# Spring Boot 3.x Maven Project for OTT Streamify Backend
# Java 17, Jar packaging, Group: com.streamy, Artifact: streamy-backend
# Dependencies: Spring Web, Spring Data MongoDB, Spring Security, Lombok, Validation, DevTools, io.jsonwebtoken, Swagger/OpenAPI

# Setup Commands
# 1. Navigate to the backend directory:
cd streamy-backend

# 2. Build and run the backend:
mvn spring-boot:run
