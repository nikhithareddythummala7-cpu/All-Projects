📋 What This Report Shows
  • Total Registered Voters: Count of all voters approved to vote across the system
  • Total Votes Cast: Actual number of votes submitted across all elections
  • Turnout Rate: Percentage of registered voters who participated = (Total Votes Cast / Total Voters) × 100
  • Election-wise Breakdown: Detailed turnout metrics for each individual election
  • Status Colors: Green (≥70% = High), Yellow (40-70% = Medium), Red (<40% = Low)
